# Skill: 算法设计

## 触发条件

当需要为问题设计算法方案时使用。

## 执行步骤

1. **问题建模**: 将自然语言描述转化为数学形式
2. **约束识别**: 输入规模、时间限制、空间限制、精度要求
3. **方案候选**: 列出 2-3 个可行算法，附带复杂度对比
4. **方案选择**: 根据约束选择最优方案，说明理由
5. **伪代码输出**: 用结构化伪代码描述算法流程
6. **正确性论证**: 用不变量或归纳法简要论证
7. **边界分析**: 空输入、单元素、最大规模、退化情况

## 输出格式

```markdown

### 问题建模

[数学形式]

### 候选方案对比

| ---- | ---------- | ---------- | -------- |

### 选定方案

[方案名] — 理由: [...]

### 伪代码
```

[伪代码]

```

### 边界条件
- 空输入: [处理]
- 最大规模: [表现]
```
